# nom-exif v3 — P5: ExifIter / Exif API surface

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Cut the v2-shaped `Exif` / `ExifIter` / `ParsedExifEntry` surface over to the v3 design from `docs/V3_API_DESIGN.md` §3.5 / §3.7 / §5.4 / §5.5: `IfdIndex` newtype, `TagOrCode`, `ExifEntry<'a>` (eager view, pub fields), `ExifIterEntry` (lazy yield, private fields + `into_result`), full `Exif` accessor matrix (`iter` / `get_in` / `get_by_code` / `gps_info` / `errors` / `has_embedded_media`), `ExifIter::{rewind, clone_rewound, parse_gps, has_embedded_media}`, and `ExifTag::from_code` + `FromStr` (with `ConvertError`).

**Architecture:** Each task is one self-contained API change with a TDD test. Public symbols are added in dependency order — newtypes (`IfdIndex`, `TagOrCode`) and minimal-coupling impls (`ExifTag::from_code`/`FromStr`) first, then the `Exif`/`ExifIter` API expansion that consumes them, then the breaking rename of `ParsedExifEntry` → `ExifIterEntry` (which forces every downstream callsite — `examples/rexiftool.rs`, `parser_async.rs` smoke tests, `exif_exif.rs::iter_to_str` — through the new `into_result` shape). Per-entry errors are *not* dropped on `From<ExifIter> for Exif`: they land in `Exif::errors()` so eager and lazy paths reach the same information. `has_embedded_media` is a static MIME-derived flag plumbed in at `parse_exif_iter` entry and threaded onto the returned iterator (HEIC/HEIF/RAF only).

**Tech Stack:** Rust 1.83, `chrono` 0.4, `bytes` 1.7.1 (already in tree from P4.5), `thiserror` 2.0.

**Phase position:** lands on `v3` branch between P4.5 (bytes unification) and P6 (track/prelude/docs). Per master plan: P4.5 is internal-only and was deliberately scheduled before P5 so that P5's diff is purely API-shape changes, not mixed concerns. Final task tags `v3-p5-done`.

**Exit criterion (verbatim from master plan P5 row):**

```rust
/// Index of an IFD (Image File Directory) within an EXIF blob.
///
/// `0` = main image (`IfdIndex::MAIN`), `1` = thumbnail (`IfdIndex::THUMBNAIL`),
/// `>=2` = sub-IFDs in the order encountered. Use the constants for the common
/// cases and [`IfdIndex::new`] for raw indexing.
#[cfg_attr(feature = "serde", derive(serde::Serialize, serde::Deserialize))]
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, PartialOrd, Ord)]
pub struct IfdIndex(usize);

impl IfdIndex {
    /// Index of the main image IFD (always `0`).
    pub const MAIN: Self = IfdIndex(0);

    /// Index of the thumbnail IFD (`1` when present).
    pub const THUMBNAIL: Self = IfdIndex(1);

    /// Construct from a raw index. `0`/`1` correspond to [`Self::MAIN`] /
    /// [`Self::THUMBNAIL`]; values `>= 2` are sub-IFDs.
    pub const fn new(index: usize) -> Self {
        IfdIndex(index)
    }

    /// Underlying raw index.
    pub const fn get(self) -> usize {
        self.0
    }
}

impl std::fmt::Display for IfdIndex {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "ifd{}", self.0)
    }
}
```

```rust
/// Either a recognized [`ExifTag`] or a raw `u16` tag code that the parser did
/// not recognize. Yielded by [`crate::ExifIterEntry::tag`] and by
/// [`crate::ExifEntry::tag`].
#[allow(unused)]
#[cfg_attr(feature = "serde", derive(Serialize, Deserialize))]
#[derive(Eq, PartialEq, Hash, Clone, Copy)]
pub enum TagOrCode {
    /// Tag is in the recognized [`ExifTag`] enum.
    Tag(ExifTag),

    /// Tag is *not* in [`ExifTag`]; raw `u16` code is preserved.
    Unknown(u16),
}

impl TagOrCode {
    /// The recognized [`ExifTag`], if any. `None` for `Unknown(_)`.
    pub fn tag(&self) -> Option<ExifTag> {
        match self {
            TagOrCode::Tag(t) => Some(*t),
            TagOrCode::Unknown(_) => None,
        }
    }

    /// The raw `u16` tag code (always available, recognized or not).
    pub fn code(&self) -> u16 {
        match self {
            TagOrCode::Tag(t) => t.code(),
            TagOrCode::Unknown(c) => *c,
        }
    }
}

impl From<u16> for TagOrCode {
    fn from(v: u16) -> Self {
        match ExifTag::from_code(v) {
            Some(tag) => TagOrCode::Tag(tag),
            None => TagOrCode::Unknown(v),
        }
    }
}

impl Debug for TagOrCode {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            TagOrCode::Tag(v) => Debug::fmt(v, f),
            TagOrCode::Unknown(v) => Debug::fmt(&format!("Unknown(0x{v:04x})"), f),

}
```

```rust
/// Lazy yield from [`ExifIter`]. Carries a *value xor error* invariant —
/// every entry holds exactly one of [`Self::value`] or [`Self::error`].
///
/// # Why private fields?
///
/// Public fields would let callers construct nonsense like `value=Some,
/// error=Some`. Private fields + getters preserve the invariant while
/// exposing the natural API: [`Self::result`] for borrowed access,
/// [`Self::into_result`] for ownership transfer (consumes `self`, no panic
/// path).
#[derive(Clone)]
pub struct ExifIterEntry {
    ifd: IfdIndex,
    tag: TagOrCode,
    res: Result<EntryValue, crate::error::EntryError>,
}

impl ExifIterEntry {
    /// IFD this entry was found in (`IfdIndex::MAIN` for the primary image).
    pub fn ifd(&self) -> IfdIndex {
        self.ifd
    }

    /// Recognized tag, or raw `u16` code if not in [`ExifTag`].
    pub fn tag(&self) -> TagOrCode {
        self.tag
    }

    /// Borrow the value. `None` iff this entry hit a parse error.
    pub fn value(&self) -> Option<&EntryValue> {
        self.res.as_ref().ok()
    }

    /// Borrow the error. `None` iff this entry parsed successfully.
    pub fn error(&self) -> Option<&crate::error::EntryError> {
        self.res.as_ref().err()
    }

    /// Borrow either value or error, mirroring the underlying invariant.
    pub fn result(&self) -> Result<&EntryValue, &crate::error::EntryError> {
        self.res.as_ref()
    }

    /// Consume self and return the value or error. No second-call panic
    /// path (the entry is moved out).
    pub fn into_result(self) -> Result<EntryValue, crate::error::EntryError> {
        self.res
    }

    pub(crate) fn make_ok(ifd: usize, tag: TagOrCode, v: EntryValue) -> Self {
        Self {
            ifd: IfdIndex::new(ifd),
            tag,
            res: Ok(v),
        }
    }
}

impl Debug for ExifIterEntry {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        let value = match &self.res {
            Ok(v) => format!("{v}"),
            Err(e) => format!("{e:?}"),
        };
        f.debug_struct("ExifIterEntry")
            .field("ifd", &self.ifd)
            .field("tag", &self.tag)
            .field("value", &value)
            .finish()
    }
}
```
